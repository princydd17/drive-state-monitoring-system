📝 Event Management

POST /api/events
Log a detection event (drowsiness, distraction, yawn)
{ "event_type": "drowsiness", "severity": "high", "duration": 5.2, "details": {"ear": 0.15, "frame_count": 20}, "user_id": "driver_001" }
GET /api/events?event_type=drowsiness&user_id=driver_001&limit=50
Retrieve detection events with filtering options

👤 Session Management

POST /api/sessions
Start a new monitoring session

PUT /api/sessions/{session_id}
